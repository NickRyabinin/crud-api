POST /books/ - CREATE

  BODY - JSON {
                  "title": title,
                  "author": author,
                  "published_at": year
              }

Все поля являются обязательными.

GET /books/ - READ

No BODY needed

PUT /books/ - UPDATE

  BODY - JSON {
                  "id": id,
                  "title": title,
                  "author": author,
                  "published_at": year
              }

Все поля являются обязательными.

PATCH /books/ - UPDATE partial

  BODY - JSON {
                  "id": id,
                  "title": title,
                  "author": author,
                  "published_at": year
              }

Поле "id" является обязательным. Остальные поля - должно присутствовать минимум одно.

DELETE /books/ - DELETE

  BODY - JSON {
                  "id": id
              }

Поле "id" является обязательным.
